# Live demo link: https://bartlomiejmazur.github.io/shopping_app/

# System Requirements

- git v2.13 or greater
- NodeJS 14 || 16 || 18
- npm v8.16.0 or greater

# How to install locally.

1. git clone https://github.com/bartlomiejmazur/shopping_app.git
2. cd .\shopping_app\
3. npm install.
4. npm start.
